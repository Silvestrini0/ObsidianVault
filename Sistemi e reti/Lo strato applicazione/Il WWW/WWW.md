(world wide web)
E' un'insieme di pagine multimediali, documenti testuali, video, collegati tra loro ai quali è possibile accedere utilizzando particolari protocolli.

Il generico schema di riferimento del web è il seguente;
Un client contatta un server http che contiene pagine web per richiedere una risorsa di cui conosce la locazione.
La risposta è la risorsa richiesta, e spesso la risorsa è un file html.
Il browser è un programma che permette di interpretare il linguaggio taggato HTML e di visualizzare la pagina web richiesta.
La pagina web può incapsulare anche programmi eseguibili localmente dal browser.
